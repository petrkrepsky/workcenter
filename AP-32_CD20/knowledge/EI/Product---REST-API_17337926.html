Enterprise Integration : Product - REST API

Created by Joachymstál Vít, last modified on Čvc 20, 2022

Předmět a výstup rekonciliace

Rekonciliační API bude vystaveno ve WSO2. Až se tak stane, některé detaily z této stránky budou odstraněny. Metoda může být přidána jako součást některé už existující REST služby na Produktovém katalogu. Zde bude upřesněno po finální implementaci v Produktovém katalogu.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

API slouží k zaslání posledních stavů produktů k okamžiku volání služby.

Posílají se produkty ve stavech Aktivní, Pozastavený, Ukončený a Odstraněný (tedy všechny stavy mimo stav Nový a Návrh).

Pomocí query parametru businessChangeDateStart lze filtrovat na požadované časové období.

Lokalizace

Toto api je lokalizované, tedy na WSO2 existují dvě verze, jedna pro Cz a druhá pro Sk.

Atributy volání

  • metoda:
    • GET /product/
  • header parametry (povinné):
    • X-Correlation-ID
  • query parametr:
    • entityType - nepovinný - specifikuje produkt - viz výčet hodnot na stránce Product - Datový model
    • businessChangeDateStart - povinný - maximálně 3 měsíce zpátky, pokud je nastaven delší čas, vrátí se bez ohledu na hodnotu parametru produkty za poslední 3 měsíce

Předmět a výstup inicializace

Zavoláním inicializace se rozumí vyvolní toku událostí, představujících aktuální obraz Produktového katalogu, tedy produktů ve stavech Aktivní, Pozastavený, Ukončený a Odstraněný (tedy všechny stavy mimo stav Nový a Návrh).

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.
