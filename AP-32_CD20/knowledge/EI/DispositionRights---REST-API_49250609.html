Enterprise Integration : DispositionRights - REST API

Created by Rak Filip, last modified on Kvě 09, 2023

Předmět a výstup rekonciliace 

Dne 14.02.2023 bylo dohodnuto, že recon API DispositionRights nebude součástí integrační analýzy v releasu IAV16. Křepský Petr 

Rekonciliační API bude vystaveno ve WSO2.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Znovu zaslání posledních změn všech entit  z časového intervalu bcd:

  • businessChangeDateStart
  • businessChangeDateEnd

Atributy volání

  • metody:
    • POST /dispositionRight/query
  • header parametry (povinné):
    • X-Correlation-ID
    • traceparent
  • query parametr (nepovinné):
    • businessChangeDateStart
    • businessChangeDateEnd
  • body
    • owner
      • crmId
    • portfolioIds
      • topasId

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace. Zavoláním inicializace se rozumí vyvolání toku všech dostupných historických událostí.

V inicializaci se datové entity ve stavu deleted nepřenášejí.

Atributy volání

  • povinné
    • X_Correlation_ID
  • nepovinné
    • location
