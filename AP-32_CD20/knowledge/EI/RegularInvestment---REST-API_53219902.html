Enterprise Integration : RegularInvestment - REST API

Created by Joachymstál Vít on Dub 27, 2023

Předmět a výstup rekonciliace

Rekonciliační API bude vystaveno ve WSO2. Až se tak stane, některé detaily z této stránky budou odstraněny.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Na vstupu lze omezit časový interval pro který se mají data poslat.

  • Časový interval je definovaný = znovuzaslání výsledných změn entit z časového intervalu
  • Časový interval není definovaný = znovuzaslání výsledných změn entit za poslední tři měsíce
  • Posílá se pouze aktuální (nejnovější) podoba každé entity

Atributy volání

  • metody:
    • POST /instrumentTransaction/query
  • header parametry (povinné):
    • X-Correlation-ID
    • traceparent
    • X-Request-ID
  • query (povinný)
    • location
  • body parametry (nepovinné):
    • businessChangeDateStart
    • businessChangeDateEnd
    • portfolioIds
      • ebrokerId
    • orderIds
      • ebrokerId
    • owner
      • crmId

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.

Zavoláním inicializačního API se rozumí vyvolání všech událostí.

Posílá se pouze aktuální (nejnovější) podoba každé entity.

Neposílají se entity se Status "deleted".

Atributy volání

  • povinné
    • X_Correlation_ID
  • nepovinné
    • location
