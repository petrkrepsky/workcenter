Atributy volání

Na vstupu lze omezit konkrétní dokument pro který lze data zaslat.

Zohlednění časových hodnot: API bude zasílat pouze aktuální informace o účtech.

Přehled metod a parametrů

  • metody:
    • GET /clientDocument
    • GET /clientDocument/{documentId}
    • GET /health (viz healthcheck endpoint)
  • header parametry (povinné):
    • X-Correlation-ID
    • traceparent
  • query parametry pro metodu GET /portfolioPosition/{documentId}
    • parentId
    • parentType
    • businessChangeDateStart
    • businessChangeDateStartEnd
  • Implementační detail
    • všechny query parametry jsou nepovinné
    • pokud je vyplněno parentId, musí být zároveň vyplněno parentType
    • pokud není vyplněno businessChangeDateStart - automaticky se bere datum 90 dní zpět od teď

Požadavky na inicializaci

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace. 

  • Do INIT/REKONCILAČNÍHO  topicu budou vloženy všechny stavy klientských dokumentů, které jsou v historii SHAREPOINT dostupné.
