Enterprise Integration : PortfolioOverview - REST API

Created by Joachymstál Vít, last modified on Lis 01, 2022

Rekonciliační API bude vystaveno ve WSO2. Až se tak stane, některé detaily z této stránky budou odstraněny.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Na vstupu lze omezit časový interval pro který se mají data poslat.

  • Časový interval je definovaný = znovuzaslání entit po poslední změně z časového intervalu, tedy pouze aktuální stavy entit
  • Časový interval není definovaný = znovuzaslání entit po poslední změně za poslední tři měsíce, tedy pouze aktuální stavy entit
  • V rekonciliaci se datové entity ve stavu deleted přenášejí.

Atributy volání

  • metoda:
    • POST /portfolioOverview/query/
  • header parametry (povinné):
    • X-Correlation-ID
    • X-Request-ID
  • query parametr (povinný):
    • location
  • body parametry (nepovinné):
    • businessChangeDateStart
    • businessChangeDateEnd
    • portfolioIds
      • ebrokerId
    • owner
      • crmId

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace. Zavoláním inicializace se rozumí vyvolání toku všech dostupných historických událostí.

V inicializaci se datové entity ve stavu deleted nepřenášejí.

Atributy volání

  • povinné
    • X_Correlation_ID
  • nepovinné
    • location
