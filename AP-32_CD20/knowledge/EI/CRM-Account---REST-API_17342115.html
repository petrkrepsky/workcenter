Enterprise Integration : CRM Account - REST API

Created by Neznámý uživatel (ex8101446), last modified by Havlena David on Pro 12, 2022

Předmět a výstup rekonciliace

Rekonciliační API bude vystaveno ve WSO2.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Znovu zaslání posledních změn všech entit  z časového intervalu bcd nebo znovuzaslání jedné datové entity dle identifikátoru

Atributy volání

  • metody:
    • POST/ Account/create/{legalType}
    • PUT/ Account/{crmIdType}/{crmId}/update
    • PATCH /Account/{crmIdType}/{crmId}/updateKomunikace
    • POST /Account/check
    • PUT /Account/{crmIdType}/{crmId}/approve
    • POST /Account/query
  • header parametry (povinné):
    • X-Correlation-ID
  • query parametr (nepovinné):
    • from
    • to
  • query parametr (povinné): 
    • mode (sync, async)
  • body
    • object obsahující:
      • AccountGUID
      • AccountKLID
      • AccountRC
      • AccountICO
      • AccountNID
      • AccountVID
      • AccountZAM

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.

Inicializace znamená:

  • zaslání všech datových entity Account, které jsou uchované v CRM, v posledním stavu
