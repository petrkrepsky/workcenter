Interface tabulky

eBroker vytvoří, plní a spravuje tyto interface tabulky:

  • Pro čerpání dat do Průběžného topicu
  • Pro čerpání dat do Inicializačního/Rekonciličaního topicu
  • Kontrolní

Každá Kafka událost sestává z Key a Value:

  • Key do sloupce KEY
  • Value do sloupce BODY

Key

Ve sloupci KEY klíč události = tradeIds.ebrokerId.
