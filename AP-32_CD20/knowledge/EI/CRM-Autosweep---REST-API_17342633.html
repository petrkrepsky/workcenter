Enterprise Integration : CRM Autosweep - REST API

Created by Neznámý uživatel (ex8101446), last modified by Rosa Tomáš on Čer 21, 2022

Požadavky na rekonciliaci

Rekonciliační API bude vystaveno ve WSO2.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Znovu zaslání posledních změn všech entit  z časového intervalu bcd nebo znovuzaslání jedné datové entity dle identifikátoru

Atributy volání

  • metody:
    • GET /Autosweep/autosweepId/{autosweepId}
    • GET /Autosweep/{crmIdType}/{crmId}?mode=async&location=xx
    • GET /Autosweep?location=xx&from=xxx&to=yyy&mode=async
  • Path parametry 
    • autosweepId identifikátor autosweep
    • kde crmIdType je z rozsahu AccountGUID, AccountKLID
    • crmId je plněn hodnotou dle crmIdType 
  • header parametry (povinné):
    • X-Correlation-ID
  • query parametr (nepovinné):
    • from (obězení výběru na datum změny entity)
    • to (obězení výběru na datum změny entity)
    • location={cz;sk} výběr pouze nastavení pro klienty cz nebo sk
  • query parametr (povinné): 
    • mode=async

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.

Inicializace znamená:

  • zaslání všech nastavení v autosweepu
