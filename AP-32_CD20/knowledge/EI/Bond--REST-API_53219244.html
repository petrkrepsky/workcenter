Předmět a výstup rekonciliace

Princip

API slouží k zaslání posledních stavů bond produktů k okamžiku volání služby.

Pomocí query parametru businessChangeDateStart lze filtrovat na požadované časové období.

Lokalizace

Toto api je lokalizované, tedy na WSO2 existuje jedna pro SK.

Atributy volání

  • metody:
    • POST /bond/query/
    • GET /health/
  • header parametry (povinné):
    • X-Correlation-ID
    • traceparent
  • body parametry (nepovinné):
    • validityDateStart
    • validityDateEnd
    • businessChangeDateStart
    • businessChangeDateEnd
    • portfolioIds
      • topasId
    • bondPriceDataId - specifikuje jednu cenu dluhopisu
    • isin - specifikuje dluhopis

Předmět a výstup inicializace

Zavoláním inicializace se rozumí vyvolní toku událostí, představujících aktuální obraz Bond katalogu.

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.
