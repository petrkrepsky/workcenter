Enterprise Integration : CRM ContactPerson - REST API

Created by Hrdlička Pavel, last modified on Bře 30, 2023

Požadavky na rekonciliaci


Specifikace rozhraní je na stránce Integrační analýzy - open api

Jedná se o nové samostatné api: contactPersonReconciliation

Atributy volání

Na vstupu lze omezit konkrétní účet pro který lze data zaslat.

Zohlednění časových hodnot: API bude zasílat pouze aktuální informace o účtech.

Přehled metod a parametrů

  • metody:
    • GET /contactPerson/
  • header parametry (povinné):
    • X-Correlation-ID
    • traceparent
  • query parametry: žádné

Vysvětlení - událostí je celkově tak malé množství (aktuálně 300-400 záznamů), že nemá význam výstup na vstupu nijak filtrovat.

Požadavky na inicializaci

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace. 

  • Do INIT/REKONCILAČNÍHO  topicu budou vloženy všechny kontakty, které jsou v historii CRM dostupné.
  • V jednotlivých událostech budou kontakty v aktuálním stavu, historie změn není evidována.
