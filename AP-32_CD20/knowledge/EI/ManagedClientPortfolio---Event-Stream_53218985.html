Popis odběru dat

Jupiter zapisuje data pomocí java API knihovny do topicu.

Změny v topiku budou zapisovány 1x měsíčně, kdy hodnoty budou obsahovat data za předchozí měsíc s granularitou 1 den.

Datová entitaSpouštěč událostiPočet událostí za denPoznámka
ManagedClientPortfolio Datum, ke kterému došlo k ocenění portfolia. Data budou dodávaná za předchozí měsíc s granularitou 1 měsíc.

JSON schémata

JSON schémata obsahují nepovinný atribut X-Correlation-ID, který se plní hodnotu header parametru stejného jména z volání init/recon API.

JSON Schéma je uloženo na stránce Integrační analýzy - schemata a formáty
