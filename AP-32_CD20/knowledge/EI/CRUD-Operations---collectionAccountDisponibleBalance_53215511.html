Výpočty, poznámky k mapování

  • instrumentID - konstanta - ID univerzálního instrumentu pro CZK - všechna prostředí: 108686 
  • accountBalances.acc - Výpočet disponibilního zůstatku (výstupní hodnota: accbalances.accBalADisponible):
    • V API AccountBalances pro všechny účty ve výstupní struktuře portfolioreports.accBalances provést:
      • pro danou měnu účtu si najít příslušnou hodnotu v MRMwebService.Models.BuyPowerInfo podle měny (MRMwebService.Models.BuyPowerInfo.Currency = portfolioReports.accBalances.accCurrency )
      • zjištění Minima (MRMwebService.Models.BuyPowerInfo.BuyPower; portfolioReports.accBalances.accBalAPresettled ) a její přiřazení do výstupu: accountBalances.accountBalanceADisponible
