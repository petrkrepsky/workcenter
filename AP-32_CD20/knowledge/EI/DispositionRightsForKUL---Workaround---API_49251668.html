Enterprise Integration : DispositionRightsForKUL - Workaround - API

Created by Rak Filip, last modified on Bře 27, 2023

Pro komunikaci s KUL(eBroker) využíváme službu  zapisPravaPristupovehoUctuKPfDK

API je vystaveno ve WSO2.

Tato služba WSDL se používá k zápisu přístupových práv k uživatelskému účtu v obchodním systému(eBroker). Umožňuje uživateli upravovat svá přístupová práva k různým portfoliím v rámci obchodního systému.

Služba využívá protokol SOAP (Simple Object Access Protocol) k výměně dat mezi klientem a serverem. Obálka SOAP obsahuje hlavičku s informacemi o kontextu zabezpečení, za kterou následuje tělo s hlavní zprávou požadavku.

Přes soap službu je možné zapsat pouze jedny práva pro jedno portfolio. Tzv jestliže v Topicu přijde pro daného uživatele více portfolio. Aplikace vybere každé portfolio a zavolá soap službu per portfolio.
