Enterprise Integration : Instrument Prices - REST API

Created by Joachymstál Vít, last modified on Bře 01, 2023

Předmět a výstup rekonciliace

Rekonciliační API bude vystaveno ve WSO2. Až se tak stane, některé detaily z této stránky budou odstraněny.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Na vstupu lze omezit časový interval pro který se mají data poslat.

  • Časový interval je definovaný = znovuzaslání výsledných změn entit z časového intervalu
  • Časový interval není definovaný = znovuzaslání výsledných změn entit za poslední tři měsíce

Atributy volání

  • metody:
    • POST /instrumentMarketPriceInstant/query/
    • POST /instrumentMarketPriceInstantDelayed/query/
    • POST /instrumentMarketPriceDaily/query/
    • POST /instrumentValuePriceInstant/query/
    • POST /instrumentValuePriceInstantDelayed/query/
    • POST /instrumentValuePriceDaily/query/
    • POST /instrumentPerformanceMarketPriceDaily/query/
    • POST /instrumentPerformanceValuePriceDaily/query/
    • POST /accruedInterest/query/
  • header parametry (povinné):
    • X-Correlation-ID
    • X-Request-ID
  • body parametry (nepovinné):
    • validityDateStart
    • validityDateEnd
    • businessChangeDateStart
    • businessChangeDateEnd
    • instrumentId - specifikuje cenný papír

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.

Zavoláním inicializačního API se rozumí vyvolání všech událostí od roku 2015. API může na svém vstupu obsahovat filtr specifikující cenný papír.

Atributy volání

  • povinné
    • X_Correlation_ID
  • nepovinné
    • Entity_Type
    • Instrument_ID
