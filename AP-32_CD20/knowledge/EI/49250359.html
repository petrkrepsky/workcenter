Enterprise Integration : History - načtení vyplněných žádostí o real-time data včetně výsledku o přístupu

Created by Havlena David, last modified by Kratochvíl Aleš on Dub 19, 2023

Operátor zobrazená data využívá k manuálnímu vyhodnocení dotazníku. Camunda řídí proces vyhodnocení dotazníku, Pokud se CRM nepodaří vyhodnotit dotazník automaticky, zavolá proces v Camundě, který spustí manuální vyhodnocení (pouze vrátí ID žádosti pro sledování stavu) Camunda pro získání historie volá CRM.
