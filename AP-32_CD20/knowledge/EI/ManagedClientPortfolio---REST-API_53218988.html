Enterprise Integration : ManagedClientPortfolio - REST API

Created by Rak Filip, last modified on Kvě 10, 2023

Rekonciliační API bude vystaveno ve WSO2. Až se tak stane, některé detaily z této stránky budou odstraněny.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Na vstupu lze omezit časový interval pro který se mají data poslat. Pro tento časový interval je definovaný "portfolioValuationDateStart" a "portfolioValuationDateEnd"

  • Časový interval je definovaný = znovuzaslání entit po poslední změně z časového intervalu, tedy pouze aktuální stavy entit

Na vstupu lze uvést unikátní indikátor jak klienta portfolio, tak samotné portfolio, tak obojí.

  • crmId -  Při zadaní pouze crmId. Jupiter vrací informace o všech portfoliích zadaného klienta. 
  • topasId - Při zadaní pouze topasId. Jupiter vrací informace o konkrétním portfoliu. 
  • crmId + topasId - Při zadaní crmId a topasId. Jupiter vrací informace o konkrétním portfoliu. 

Atributy volání

  • metoda:
    • POST /ManagedClientPortfolio/query/
  • header parametry (povinné):
    • X-Correlation-ID
    • X-Request-ID
  • query :
    • businessChangeDateStart
    • businessChangeDateEnd
  • body parametry:
    • portfolioValuationDateStart
      • regex  "^(2021-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01]))|([2-9]\d{3}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01]))$" 
      • Atribut portfolioValuationDateStart pro ocenění portfolia ve formátu „YYYY-MM-DD“ má povolená hodnotu, která musí být větší než 1. ledna 2021.
    • portfolioValuationDateEnd
      • regex  "^(2021-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01]))|([2-9]\d{3}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01]))$"  
      • Atribut portfolioValuationDateEnd pro ocenění portfolia ve formátu „YYYY-MM-DD“ má povolená hodnotu, která musí být větší než 1. ledna 2021.
    • owner
      • crmId
    • portfolioIds
      • topasId

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace. Zavoláním inicializace se rozumí vyvolání toku všech dostupných historických událostí.
