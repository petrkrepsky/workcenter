Enterprise Integration : CRM AuthorizedAccount - REST API

Created by Hrdlička Pavel, last modified on Pro 21, 2022

Předmět a výstup rekonciliace

Rekonciliační API bude vystaveno ve WSO2 - v rámci api DigiClientOperations - jako samostatná metoda s níže uvedenými vlastnostmi.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Znovu zaslání posledních změn všech entit  z časového intervalu bcd nebo znovuzaslání jedné datové entity dle identifikátoru

Atributy volání

  • metody:
    • POST /AuthorizedAccount/query
  • header parametry (povinné):
    • X-Correlation-ID
  • query parametr (nepovinné):
    • from
    • to
  • query parametr (povinné): 
    • mode (sync, async)
  • body
    • object obsahující:
      • accountGUID
      • accountKLID

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.

Inicializace znamená:

  • zaslání všech datových entity AuthorizedAccount, které jsou uchované v CRM, v posledním stavu
