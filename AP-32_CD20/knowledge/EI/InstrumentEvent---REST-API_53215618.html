Enterprise Integration : InstrumentEvent - REST API

Created by Joachymstál Vít, last modified on Bře 22, 2023

Předmět a výstup rekonciliace

Rekonciliační API bude vystaveno ve WSO2. Až se tak stane, některé detaily z této stránky budou odstraněny.

Specifikace rozhraní je na stránce Integrační analýzy - open api

Princip

Na vstupu lze omezit časový interval pro který se mají data poslat.

  • Časový interval (BCD) je definovaný = znovuzaslání výsledných změn entit z časového intervalu
  • Časový interval (BCD) není definovaný = znovuzaslání výsledných změn entit za poslední tři měsíce

Atributy volání

  • metody:
    • POST /instrumentEvent/query/
  • header parametry (povinné):
    • X-Correlation-ID
    • traceparent
    • X-Request-ID
  • body parametry (nepovinné):
    • businessChangeDateStart
    • businessChangeDateEnd
    • validityDateStart
    • validityDateEnd
    • instrumentIds
      • ccpId
    • eventType

Předmět a výstup inicializace

Inicializační API může, ale také nemusí, být REST služba. Záleží na konkrétní realizaci preferované vlastníkem aplikace.

Zavoláním inicializačního API se rozumí vyvolání všech událostí od roku 2015. API může na svém vstupu obsahovat filtr specifikující cenný papír (instrumentIds.ccpId) a/nebo druh události (eventType).

Atributy volání

  • povinné
    • X_Correlation_ID
  • nepovinné
    • Event_Type
    • Instrument_ID
