Disponibilní zůstatek

Jedná se o disponibilní zůstatek pro výběr peněz ze sběrného účtu na běžný účet.

Pojem disponibilní zůstatek se používá také v kontextu pro obchodování a ten je zjišťován jiným způsobem.

  • získání hodnoty PreSetled zůstatků na všech sběrných účtech daného portfolia (vstup) v měně sběrného účtu a referenční měně (vstup, možnosti CZK, EUR, USD)
  • hodnotu BuyPower daného portfolia  ve všech definovaných měnách v MRM
  • výpočet disponibilního zůstatku pro všechny sběrné účty v dané měně: jako minimum z presetledZůstatku a ptfBuyPowerTotal.


Předpoklady

  • změnový požadavek na eBroker (S1) na změnu výstupu API Trading, GET /AccountBalances - API by měla vracet ID portfolia z obou systémů - Topas/eBroker. V atributu  portfolioID  by měla vracet Id portfolia z Topasu, v portfolioID2 by měla vracet  ID portfolia z eBroker. V současné verzi API vrácí ID portfolia toho systému, který byl použitý na vstupu v atributu referenceSystem.
